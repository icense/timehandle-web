개인정보 처리방침

최종 업데이트: 2026년 5월 1일

🔒 핵심 약속: TimeHandle은 개인정보를 수집하지 않습니다
등록 없음, 로그인 없음, 서버 업로드 없음. 모든 데이터는 사용자의 휴대폰에만 저장됩니다.

1. 데이터 수집

TimeHandle은 어떠한 개인정보도 수집하지 않습니다. 자체 서버가 없으며 제3자에게 데이터를 전송하지 않습니다.

  • ❌ 이름, 이메일, 전화번호 등의 개인정보를 수집하지 않습니다
  • ❌ 기기 식별자(IDFA, IDFV)를 수집하지 않습니다
  • ❌ 위치 정보를 수집하지 않습니다
  • ❌ 사용 통계 데이터를 수집하지 않습니다

2. 캘린더 데이터

TimeHandle은 핵심 기능을 제공하기 위해 시스템 캘린더 읽기 권한이 필요합니다. 모든 캘린더 데이터:

  • ✅ iPhone/iPad에만 로컬 저장됩니다
  • ✅ 서버에 업로드되지 않습니다
  • ✅ 제3자와 공유되지 않습니다
  • ✅ 앱 삭제 시 자동으로 삭제됩니다

iOS 시스템 설정에서 언제든지 캘린더 접근 권한을 철회할 수 있습니다. 철회 후 앱은 캘린더를 읽을 수 없게 됩니다.

3. 구매 정보

앱 내 구매 및 구독은 Apple StoreKit을 통해 처리됩니다:

  • Apple이 모든 결제 정보를 처리합니다. TimeHandle은 사용자의 결제 정보를 획득할 수 없습니다
  • 앱은 StoreKit API를 통해 구독 상태만 확인하며 구매 기록을 저장하지 않습니다
  • 확인은 사용자 기기에서 로컬로 수행되며 제3자 서버를 거치지 않습니다

4. 알림 권한

알림 권한을 활성화하면 리마인더가 사용자 기기에서 로컬로 생성됩니다. 모든 리마인더 로직은 기기에서 실행되며 데이터 전송이 발생하지 않습니다.

5. 제3자 서비스

TimeHandle은 제3자 분석, 통계, 충돌 보고 서비스를 사용하지 않습니다. 광고 SDK도 통합되어 있지 않습니다.

  • Google Analytics 없음
  • Firebase 없음
  • Facebook SDK 없음
  • 어떠한 추적이나 분석도 없음

6. 데이터 삭제

다음 방법으로 모든 데이터를 삭제할 수 있습니다:

  • TimeHandle 앱 삭제: 모든 로컬 데이터가 즉시 삭제됩니다
  • iOS 설정에서 캘린더 권한 철회: 앱이 캘린더에 접근할 수 없게 됩니다

서버 데이터를 저장하지 않으므로 별도의 삭제 요청이 필요하지 않습니다.

7. 아동 개인정보

TimeHandle은 아동을 포함한 어떠한 사용자 정보도 수집하지 않습니다. 모든 사용자가 안전하게 이용할 수 있습니다.

8. 개인정보 처리방침 변경

본 개인정보 처리방침을 업데이트하는 경우 앱 내 또는 웹사이트에 변경 사항을 게시합니다. 연락처 정보를 수집하지 않으므로 정기적으로 이 페이지를 확인하시기 바랍니다.
